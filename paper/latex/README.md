# LaTeX template (optional)

This folder contains a minimal LaTeX template you can use to produce a PDF version of the ScrubID paper.

## Compile

From this directory (`paper/latex/`):

1. `pdflatex main.tex`
2. `bibtex main`
3. `pdflatex main.tex`
4. `pdflatex main.tex`

The bibliography is shared with the pack (`bib/references.bib`).
